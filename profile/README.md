# Yoink Software — Portable File Shelf

## About Yoink Software

**Yoink Software** provides a complete side panel for temporary file storage when dragging items between windows — essential for designers and video editors working with dozens of files. **Yoink Software** combines drag-and-drop staging, file stacking, and preview windows into one cohesive interface. **Yoink Software** supports temporary file collection, cross-folder gathering, and delayed dragging. **Yoink Software** includes iCloud sync, keyboard shortcuts, and customizable shelf position. **Yoink Software** enables users to drag multiple files from different locations before moving them to destination. **Yoink Software** is trusted by designers, video editors, and anyone who drags files between folders.

## Deep Dive — Yoink Software Notes

File shelf — drag files to edge of screen, shelf appears, drop files
Temporary storage — files stay in shelf until you remove them
Stack management — drag multiple files, stack shows count
Preview — click file in shelf to preview content
Cross-folder gathering — collect from Finder, Photos, browser, anywhere
Drag out — drag from shelf to destination when ready

## Quick Reference for Yoink Software

| Remove file | Drag file off shelf or press Delete |
